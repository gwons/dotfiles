Installation
------------

This dotfiles only supports MacOS, Windows, and Ubuntu.

  1. Clone the repository:

    git clone --depth 1 git@github.com:seonggwonyoon/dotfiles.git ~/.dotfiles

  2. Install dotfiles

	~/.dotfiles/install

  3. Enjoy your shell!
