Installation
------------

This dotfiles only supports MacOS, Windows, and Ubuntu.

  1. Clone the repository:
    ```console
    git clone git@github.com:seonggwonyoon/dotfiles.git "~/.dotfiles"
    ```

  2. Execute setup.sh
    ```console
	~/.dotfiles/setup.sh
	```

  3. Enjoy your shell!
